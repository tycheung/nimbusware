# v1.1 ship checklist (finish line)

Operator runbook to close the last **5/6 → 6/6** finish-line gate. Code paths are shipped; this checklist covers **distribution publish** and **soak verification**.

## One-command local gates (build-only)

From repo root with Poetry env active:

```bash
poetry run python scripts/run_v11_ship_gates.py
```

This runs:

1. Bootstrap publish workflow contract tests
2. VS Code extension publish workflow contract tests
3. Bootstrap wheel **build + twine check** (no upload)
4. VS Code extension **VSIX package** (no upload)
5. `measure_intent_to_patch.py --via-chat` → updates `benchmarks/latest_intent_to_patch.json`
6. `run_live_writers_soak.py` → writes `benchmarks/latest_live_writers_soak.json`

Skip benchmarks for a faster distribution-only pass:

```bash
poetry run python scripts/run_v11_ship_gates.py --skip-benchmarks
```

## P0 — PyPI bootstrap (first publish)

Prerequisites: [pypi-publish.md](pypi-publish.md), `PYPI_API_TOKEN` in GitHub secrets.

**Option A — GitHub Actions (recommended)**

1. Open **Actions → Publish bootstrap wheel → Run workflow**
2. First run: leave `publish_pypi` **false** (build-only validation)
3. When ready: set `publish_pypi=true` and ensure `PYPI_API_TOKEN` is set
4. Verify: `pip install nimbusware-bootstrap && nimbusware-bootstrap --print-only`

**Option B — Local**

```bash
export PYPI_API_TOKEN=...
poetry run python scripts/run_v11_ship_gates.py --skip-benchmarks --publish-pypi
```

TestPyPI first (optional):

```bash
export TESTPYPI_API_TOKEN=...
poetry run python scripts/publish_bootstrap_release.py --testpypi
```

## P0 — VS Code Marketplace (first publish)

Prerequisites: [vscode-marketplace.md](vscode-marketplace.md), publisher `nimbusware`, `VSCE_PAT` in secrets.

**Option A — GitHub Actions**

1. **Actions → Publish VS Code extension → Run workflow**
2. Build-only: `publish_marketplace=false` (downloads VSIX artifact)
3. Release: `publish_marketplace=true`

**Option B — Local**

```bash
export VSCE_PAT=...
poetry run python scripts/run_v11_ship_gates.py --skip-benchmarks --publish-vscode
```
